Readme

Lists are the foundation of Cena v2 - Create a list for your shopping trip, or even for a recipe. Lists are recursive, ie lists can contain other lists.

To add a recipe to Cena v2 create a new list (Lists -> Add new), and fill out the relevant info. Remember to tag new recipes symantically:

  • 'grocery' defines a new grocery list. This should be used by convention so that grocery lists are searchable in the search box (it will look for this tag).
  • 'recipe' defines a new recipe. Lists with this tag are marked in the overall list view with a light blue header to help them stand out.
  • 'no-gluten' defines a gluten-free recipe. It will put a [GF] into the toolbar.
  • 'veg' defines a vegitarian meal. It will put a [V] into the toolbar.
  • 'maggie' defines a maggie-freindly meal. It will put a [MG] into the toolbar.
  • 'ryan' defines a ryan-freindly meal. It will put a [RY] into the toolbar.
  • 'bob' defines a bob-freindly meal. It will put a [B] into the toolbar.
To sort lists into groups, tag them with tags that start with sort-. So, sort-abc gets matched with all other sort-abcs and not with other tags. Anything that doesn't provide a sort-* tag is put into the unsorted group.

To set up stores, tag them with tags that start with shop-; This means shop-abc becomes a store that can easily be selected for an item by clicking the button.

If you want any more tags, add them yourself! Settings -> Tags. Type in your tag and hit enter in the box at the bottom of the list.

To add anything into a list, type its name into the textbox above the list and click add. Remember, you can add lists to lists, so list names will also work here. If you add a list and you only want a few things from it, a list of all the components of the list will show up allowing you to delete some unneeded ingredients.

Descriptions should be a short phrase describing the item.
